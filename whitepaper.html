Overview

CivicNet is an open-source, CPU-mineable Proof-of-Work cryptocurrency, built for accessibility over exclusivity. The full whitepaper covers the project's current implementation — the civiclight v2 ASIC-resistant algorithm, tokenomics, and live infrastructure — alongside the finalized design for CivicNet's next major upgrade: Hybrid Proof-of-Work + Proof-of-Stake.

What's inside

  • How civiclight v2 achieves real ASIC resistance, and how the time-based hard fork activation works
  • Full tokenomics: supply, halving schedule, and the transparent 1% developer allocation
  • The technical design of Hybrid PoW+PoS — a Blackcoin-inspired staking model with no minimum balance and whale-resistant, non-linear staking odds
  • Consensus-level specification: kernel hash, stake modifier, coinstake transaction structure

Status

Sections on the live network (algorithm, tokenomics, infrastructure) describe CivicNet as it currently runs. The Hybrid PoW+PoS section describes a finalized design direction — not yet implemented in code. Any consensus change of this scope will be extensively tested on a public testnet before mainnet activation.
