Frequently Asked Questions

Answers to the most common questions about CivicNet.

What is CivicNet?

CivicNet is an open-source cryptocurrency built to provide a simple, transparent and community-driven blockchain. The network currently uses the custom civiclight Proof-of-Work algorithm and is designed to keep mining accessible on ordinary CPUs.


What algorithm does CivicNet use?

CivicNet uses the custom civiclight algorithm. It is lightweight and CPU-friendly, allowing users to mine without specialized ASIC hardware.


Is CivicLight ASIC resistant?

Not yet. Today's civiclight algorithm is CPU-friendly but not fully ASIC-resistant. A future version of the algorithm with stronger ASIC resistance is planned as part of the long-term roadmap.


How can I start mining?

Download the official CPU miner, create a CivicNet wallet, then connect to the official mining pool or mine directly using your own node.


Can I solo mine?

Yes. You can connect your miner directly to your own CivicNet node using RPC.


What is the current block reward?

The current reward is 77 CIVIC per block. Block rewards halve every 2,100,000 blocks.


What is the total supply?

The maximum supply is approximately 315 million CIVIC. No additional coins will ever be created beyond the scheduled emission.


Was there a premine?

Yes. A transparent 1% developer premine (3,150,000 CIVIC) was created in Block 1 to fund long-term development, infrastructure, community initiatives and project maintenance. The remaining 99% of all coins are distributed through mining.


Is CivicNet open source?

Yes. All source code is publicly available on GitHub, allowing anyone to review, audit and contribute to the project.


Where can I get help?

Visit the official GitHub repository or join the community channels for support, bug reports and development discussions.
